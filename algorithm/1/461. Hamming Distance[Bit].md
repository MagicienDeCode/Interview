# 461. Hamming Distance

# Main Idea
1. xor ^, same 0, diff 1
2. x xor y, then from 0 to 31, & 1, count results

```java
class Solution {
    public int hammingDistance(int x, int y) {
        int xXorY = x ^ y;
        int count = 0;
        for(int i = 0; i < 32; i ++){
            if((xXorY & 1) == 1){
                ++count;
            }
            xXorY >>= 1;
        }
        return count;
    }
}
```
